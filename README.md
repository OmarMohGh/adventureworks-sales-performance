# AdventureWorks Sales Performance Report

An end-to-end Power BI project analyzing $24.9M in sales data to track revenue, profit margins, and product performance for a global manufacturer.

## Project Overview
This dashboard transforms raw sales and customer data into a strategic tool for executive leadership. It provides a high-level summary of financial health while allowing for granular analysis of product subcategories, customer demographics, and regional trends.

## Dataset
The analysis is based on seven interconnected data files from the AdventureWorks database:
* **Sales_data.csv:** Core transactional data including revenue and costs.
* **Product_data.csv:** Details on product categories (Bikes, Components, Clothing, Accessories).
* **Customer_data.csv:** Profiles including income levels and occupations.
* **Sales Territory_data.csv:** Geographic sales regions across North America, Europe, and Pacific.
* **Date_data.csv:** Calendar table for time-intelligence calculations.
* **Reseller_data.csv:** Information on B2B sales partners.
* **Sales Order_data.csv:** Metadata regarding individual customer orders.

## Tools Used
* **Power BI:** Data modeling and interactive visualization.
* **Power Query:** Merging multiple CSV files, data cleaning, and data type transformation.
* **DAX:** Implementation of complex measures for Total Profit, Revenue Goals, and Year-over-Year (YoY) growth.

## Key Insights
* **Revenue Performance:** Total revenue reached $24.9M, successfully hitting the target goal.
* **Profitability:** Maintained a strong 42.1% profit margin ($10.5M total profit).
* **Product Trends:** Accessories and Tires/Tubes are the volume leaders, while Bikes remain the primary revenue driver.
* **Demographics:** "Management" and "Professional" customer segments show the highest purchase frequency.
* **Regional Success:** Active tracking of six countries, identifying top-performing territories.
